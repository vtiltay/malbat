# Malbat.org - Collaborative Genealogy Platform

Modern collaborative genealogy platform for building and sharing family trees with Gramps integration for importing genealogical data.

## What It Does

Malbat.org is a collaborative genealogy platform allowing users to build, manage, and share their family trees with comprehensive features for organizing family information and historical data.

## Key Features

- Complete family tree building and visualization
- Photo and document management
- Family collaboration tools
- Data sharing controls and privacy settings
- Chronological event timeline
- Family tree search functionality
- GEDCOM format export
- Family update notifications
- Gramps import functionality
- Modification proposal system
- Validation workflows
- Multiple user roles and permissions

## Technology Stack

- **Backend**: Django 4.2+, PostgreSQL
- **Frontend**: Bootstrap, JavaScript
- **API**: Django REST Framework (optional)
- **Integration**: Gramps genealogical data support
- **Server**: Gunicorn, Nginx

## Project Structure

```
├── config/               # Django settings
├── genealogy/            # Family tree management
├── accounts/             # User profiles
├── media/                # Document/photo storage
├── api/                  # REST API (optional)
├── templates/            # HTML templates
└── static/               # CSS, JS, images
```
